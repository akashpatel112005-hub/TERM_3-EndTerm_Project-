## 📌 Problem Statement

Engineering students often face difficulties during exam preparation due to:

- Scattered study resources (notes, videos, PDFs)  
- Lack of structured learning paths  
- Ineffective revision techniques  
- No instant doubt-solving support  

### ⚠️ Consequences

- Time wasted switching between resources  
- Overwhelm due to large syllabus  
- Last-minute cramming instead of consistent study  

---

## 🎯 Target Users

- Undergraduate engineering students (2nd–4th year)  
- Students preparing for semester exams  
- Self-learners studying technical subjects  

---

## 💡 Solution: AI Study Companion

A smart web application that combines:

- 📚 Structured Learning  
- 🧠 Active Recall via Quizzes  
- 🤖 Real-time AI Tutoring  

---

## 🧩 Key Features

### 🗺️ AI Study Path Generator
- Breaks topics into step-by-step learning modules  
- Creates structured study plans  
- Encourages consistency  

### 📝 AI Quiz Generator
- Dynamically generates quizzes  
- Promotes active recall learning  
- Tracks performance over time  

### 🤖 AI Tutor
- Ask doubts anytime  
- Get simplified, instant explanations  
- Acts like a personal mentor  

### 📊 Dashboard
- Track learning progress  
- View completed topics  
- Analyze quiz performance  

### 🔐 Authentication System
- Secure Signup/Login  
- Protected Routes  
- Personalized user data  

### 💾 Persistent Data (CRUD)
- Save/update study plans  
- Store quiz history  
- Track long-term progress

2. Accesses dashboard  
3. Generates study plan  
4. Studies topics step-by-step  
5. Takes quizzes  
6. Tracks progress  
7. Uses AI tutor for doubts
